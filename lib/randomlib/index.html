Randomlib

RandomLib is a C++ interface to the Mersenne Twister random number generator MT19937 and to the SIMD-oriented Fast Mersenne Twister random number generator, SFMT19937. It provides access to random integers and reals at a variety of precisions. RandomLib also contains new algorithms which permit exact sampling from the exponential and normal distributions (provided that the underlying generator is perfect).
